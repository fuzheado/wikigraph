wikigraph

Explore how Wikipedia articles connect

What it does

wikigraph builds interactive connection graphs from Wikipedia articles. It shows how articles link to each other — through direct wikilinks, shared categories, and shared named entities (people, places, organizations). Each article is colored by topic cluster (Sports, Music, Politics, etc.).

Data sources

  • Top Articles — daily top 100 most-viewed from the Hatnote API
  • Custom List — paste your own article titles (one per line)
  • PagePile — import a list by PagePile ID
  • Category — fetch articles from a Wikipedia category (with subcategory depth)

Article metadata (categories, wikilinks, extracts, images) comes from the MediaWiki API. spaCy NLP extracts named entities from article summaries.

Controls

  • Top Articles / Custom List — switch modes
  • Search — filter visible articles by name
  • Spacing — adjust node repulsion
  • Helpers / Labels / Legend — toggle visual layers
  • 📷 Article / Wikidata — switch thumbnail source
  • ▶ Play — auto-advance through articles
  • ⟳ Refresh — clear cache and rebuild
  • Ignore / Hide — exclude articles or groups
  • 🔗 Share — copy a bookmarkable URL of the current graph
  • Hover — highlight connections
  • Click an article — side panel with summary and connections

Tech

Python pipeline (httpx, NetworkX, spaCy) + D3.js v7 frontend. Top Articles data from the hatnote/top project. PagePile from pagepile.toolforge.org.
