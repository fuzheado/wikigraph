wikigraph

Interactive Wikipedia connection graphs

How it works

Articles are fetched from the Hatnote API (Top Articles mode) or entered directly (Custom List mode). Each article is enriched with categories, wikilinks, and extracts from the MediaWiki API. spaCy NLP extracts named entities (people, organizations, places) from the summaries.

Controls

  • Top Articles — daily top 100 from Hatnote API
  • Custom List — paste your own article titles
  • Search — filter articles by name
  • Ignore — exclude specific articles
  • Spacing — adjust node repulsion
  • Play — auto-advance through articles
  • ⟳ Refresh — clear cache and rebuild
  • Hover — highlight connections
  • Click — article details + connected articles

Data

Built with Python (httpx, NetworkX, spaCy), D3.js v7, and Wikipedia APIs. Top-100 data from the hatnote/top project.
